# Backend

### 첫번 째 시간

1. 서버 기초 개념 
   - 웹사이트에 들어갈 때 일어나는 일들
   - 서버의 역할 
   - 인터넷에 대해서 
   - HTTP (Protocol에 대해서)
   - 그래서 우리가 하려는 것 (서버 개발에 대한 이야기)
2. 개발 환경 설정
   - node js, vscode 
### 두번째 시간

1. 직접 code 짜보기 
   - JS를 이용 (JS 기본 문법 C와 비교해서 설명)
   - 화면에 "<h1>Hello, world</h1>" 띄우기 
   - 프론트엔드 시간에 제작한 웹 페이지 화면에 띄우기
2. 중간 설명
   - 모듈(http, fs, require()),  - 간단하게 모듈이 뭔지
   - __dirname -> console로 찍으면서 설명
   - request, response 복습 설명
   - port 간단 설명
